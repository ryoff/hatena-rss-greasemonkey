# Hatena rss GM(GreaseMonkey)

## 概要
* 各サイトにrssショートカットを表示します。
    * rssははてなブックマーク経由して登録します。
    * [はてぶで20以上のブックマークを得た新着記事だけのrss]など指定できます。
